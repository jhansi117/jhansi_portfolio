    What are Arrays?

    Array is a list of items belonging to same datatype and the elements are stored in contiguous memory location.

    Operations in an array:

    1. 1.Create : arr = []
    2. 2.Input : arr.insert(index,element) or arr.append(element)
    3. 3.Output : print(arr)
    4. 4.Update : arr[0] = 5
